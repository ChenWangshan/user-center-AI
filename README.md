# user-center

`user-center` is a Spring Boot backend service template for user domain capabilities.

## Features

- Independent Maven project
- User center health check endpoint
- Demo user profile endpoint
- Ready for further business module expansion

## Run

```bash
mvn spring-boot:run
```

## Endpoints

- `GET /api/user-center/health`
- `GET /api/user-center/profile/demo`

## 接口文档访问地址

- http://localhost:8081/swagger-ui.html
- http://localhost:8081/v3/api-docs
